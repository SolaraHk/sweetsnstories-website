# START HERE — Sweets N Stories Website Build Pack

This repository is a handoff pack for the Sweets N Stories cake shop website build.

## What to do first

1. Read `project-build-prompt.md` in full.
2. Inspect every image in `references/` before writing code:
   - `design-01-home.png` — homepage direction.
   - `design-02-shop-collection.png` — shop/category grid direction.
   - `design-03-product-detail.png` — product page/order form direction.
3. Treat the screenshots as the primary visual reference. Match their layout, spacing, typography, palette, card rhythm, product detail form, footer style, and desktop/mobile proportions closely.
4. Build the actual website files in this repo. Do not stop at a plan.
5. Run the site locally, inspect it in browser on desktop and mobile widths, fix obvious mismatches, then commit your changes.
